# How to run

**Run puter**
```bash
git clone https://www.github.con/DarkCodeStudios/Puter-app
cd Puter-app
npm install
npm start
```
(do not open http://puter.localhost:4100/)

**Run app**
```bash
cd app
npm install
npm start
```
